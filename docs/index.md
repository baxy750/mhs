
# Magyar három szóban

Welcome to this resource for learning Hungarian using simple sentence structures.

## Sentence Example

Hungarian:  
Élek {{ morph("Magyar", "-ország", "-ban") }}

English:  
I live in Hungary.

Notes:  
- **Élek** = I live  
- **Magyarország** = Hungary  
- **-ban** = in
